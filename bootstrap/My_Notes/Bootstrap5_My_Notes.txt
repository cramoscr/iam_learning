[ ] Bootstrap overview

    + Bootstrap was originally created by Twitter

    + Provides UI (front-end) resources for web development.

    + Can be used in a web page:
        a. Referencing  a CDN repository
        b. Downloading the CSS and JS files and including them
           into the web page source folder
           
    + For SaaS purposes, can be installed on a server by using
          npm
          gem

[ ] How to use bootstrap

    + Alternative 1.

        Incluye a <link rel=xxx> tag in the html header
        section pointing to a CDN

    + Alternative 2.

        Incluye a <link rel=xxx> tag in the html header
        section pointing to local file where you
        downloaded the CSS and JS bootstrap files
